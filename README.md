# 🌦️ Weather 3D App

A modern, immersive **3D weather application** built with React, delivering real-time weather updates with stunning visuals, smooth animations, and dynamic day/night effects.

🔗 **Live Demo:** [https://weather3dapp.vercel.app/](https://weather3dapp.vercel.app/)
🔗 **Repository:** [https://github.com/Sujon-Ahmed/weather-3d-app](https://github.com/Sujon-Ahmed/weather-3d-app)

---

## ✨ Features

* 🔍 Search weather by city name
* 🌡️ Real-time weather data (temperature, conditions, etc.)
* 🌙 Dynamic **day/night cycle** based on real-time location
* 🌧️ Weather-based animations (rain, clouds, sunlight)
* 🎨 Modern UI (inspired by iOS weather & Dribbble designs)
* ⚡ Smooth 3D rendering experience
* 📱 Fully responsive (mobile + desktop)

---

## 🚀 Tech Stack

* **React + TypeScript**
* **Vite**
* **Three.js / React Three Fiber**
* **Weather API (OpenWeather or similar)**
* **CSS / TailwindCSS**

---

## 📦 Installation & Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/Sujon-Ahmed/weather-3d-app.git
cd weather-3d-app
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## ⚙️ Environment Variables

Create a `.env` file in the root directory and add your API key:

```env
VITE_WEATHER_API_KEY=your_api_key_here
```

---

## 🧠 How It Works

1. User searches for a city
2. App fetches real-time weather data from API
3. Weather conditions dynamically update:

   * Scene lighting (day/night)
   * Weather effects (rain, clouds, sun)
4. 3D environment renders changes smoothly
